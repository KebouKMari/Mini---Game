***************************************Game Plus/Minus - For Beginners - language C *************************************

Hi friends !!!!

This is a sample code of a little game that you can program yourself 
if you are a beginner in C programming.

Principle: Try to guess the word in machine (single player mode) or the one entered by another person (two player mode) 
two players) with as few tries as possible.

Warning: - Enter only whole numbers otherwise you may have infinite loops (no special characters or letters).
                - For the two player mode, enter a mystery number that belongs to the level you have chosen; otherwise BAM! :(

How it works: Open your Ide(codeblocks), then open the following file: "MyFirstProgramC.cbp" you will see the source code of the program
and you can compile it to see the result.
